# Smart_Shopping_Cart

## Overview
The Smart Shopping Cart System utilizes an ESP8266 microcontroller and RFID technology to streamline the shopping experience. The system automatically detects products added to or removed from the cart using RFID tags. Each product's details are displayed on an LCD, and the total cost is updated in real-time. This system aims to reduce checkout time and improve the overall shopping experience.

## Features
- Real-time product detection using RFID
- Automated cost calculation
- LCD display for product details and total cost

## Hardware Requirements
- ESP8266 microcontroller
- RFID tags and reader
- LCD display
- Push buttons
- Power supply

## Software Requirements
- Arduino IDE
- ESP8266WiFi library
- LiquidCrystal_I2C library
